# worldcup

Modern Next.js App Router starter using Next.js 16, React 19, TypeScript, Tailwind CSS, and shadcn/ui.

## Scripts

```bash
npm run dev
npm run lint
npm run typecheck
npm run build
```

## Structure

- `src/app` for App Router routes and layouts
- `src/components/ui` for shadcn/ui primitives
- `src/components/shared` for reusable app components
- `src/hooks` for reusable hooks
- `src/lib` for generic utilities
- `src/types` for shared TypeScript types
